# 知识调用流程

当你不是想“复习一本书”，而是想解决一个现实问题时，用这个流程。

## 输入

写下当前问题：

```text
我现在要处理的问题：
我希望得到的输出：
限制条件：
可接受的启发范围：
```

## 调用路径

1. 先查 `03-topics/topic-index.md`，找到相关主题。
2. 再查 `02-cards/`，优先调用方法卡和案例卡。
3. 如果主题索引不够，再从微信读书按关键词搜索笔记。
4. 输出时保留来源书名、作者、原始划线或想法位置。

## 输出格式

```text
问题：

可调用观点：
1.
2.
3.

可使用方法：
1.
2.

相关案例：
1.
2.

我会如何应用到当前问题：

还需要补充阅读：
```
